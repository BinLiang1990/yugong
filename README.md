## CodingFarmes
愚公码农😄

这是一个纯前端的低代码解决方案,包含核心功能:

- 可视化编辑器高度可订制(页面、组件)
- EventEmitter 事件触发与事件监机制,用于统一管理业务组件与全局的事件
- runningTimes 业务组件之间共享数据
- Api配置 与数据建立连接
- 模板化管理,按需加载
